准备：
1.请确定环境变量的java home已经正确设置
2.确保javaw指令可以正常使用
3.run.bat运行代码生成器
4.程序运行时会在logs文件夹下生成今天的日志文件
5.htmlDemoGenerate.bat运行html静态页面生成功能

代码生成器的使用：
1.填写或选择对应属性：
 类包路径：要生成的代码的类包前缀，如要生成com.wonders.user.bo下的UserEntity类，则com.wonders为类包路径
模块名称：要生成的代码所属的模块，建议首字母小写
输出编码：生成的java文件和jsp文件的编码方式
类包输出路径：java代码文件输出路径，一般是工程的src文件夹
web输出路径：jsp文件的输出路径，一般是工程的web文件夹
表说明：数据表的说明，会在jsp显示为页面标题
选择输出：选择需要生成的代码
是否生成：如果不生成，则此属性在所有生成的代码中不会出现
属性说明：属性的中文解释，会在jsp显示为该属性的名字
编辑类型：jsp编辑页面的控件类型，don't display为不显示在jsp的编辑页面
是否必填、验证属性：属性的验证规则
列表显示：是否在jsp列表页面中显示

静态deml生成器的使用：
1.excel下存放data.xls数据文件，这里 文件的格式、内容格式和名字需要和例子相同
2.生成的文件存放在htnlDemos文件夹下

关于模板：
模板是用freemarker编写的，所有模板在templete文件夹下
